INTRODUCTION

The NWS Wiring user interface is a set of desktop ‘thick-client’ applications.


The UI design is a drawing-tree and tab-based design that is based on existing UI frameworks for a number of Microsoft data-intensive editing products, including: Visual Studio 2010, SQL Server Management Studio, and Expression Blend 3.0.

 

The Wiring application user interfaces is built using Windows Presentation Foundation (WPF).
