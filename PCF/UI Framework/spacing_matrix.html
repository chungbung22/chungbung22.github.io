SPACING MATRIX

INTRODUCTION

Spacing is very important when establishing a consistent look and feel. Spacing in the past was hardcoded, which lead to pages appearing very disorganized and cluttered. Below, we have defined the different types of spacing available, and provided a matrix of when best to use the different sizes.

Spacing

Size

When to use

spacing-xs

5px

Between icon and label

spacing-sm

10px

Between label and field

spacing-md

15px

Between stacked buttons or labels

spacing-lg

20px

Between related sections of a page

spacing-xl

35px

Between top of page and first element, or between separate sections of a page

spacing-xxl

70px

Between last element and bottom of page
