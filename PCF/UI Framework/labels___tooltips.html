LABELS & TOOLTIPS

LABELS

• Top-align labels with multi-line fields


• Right justify labels for fields read as columns


• Label all panels, tables, pages, columns, rows, input boxes.


• Display user’s language for labels rather than use database

    names


• Add object type to tab labels

TOOLTIPS

• Top-align labels with multi-line fields


• Tooltip should appear above the label or input field upon hover


• Tooltip should have max character length: 100

INTRODUCTION

Labels and in-field prompts are short descriptions that identify what information is being requested for input controls.


When the relationship between multiple input controls needs to be clarified, use a label


When there is limited space, use a tooltip to provide additional info if necessary

Labels are top-aligned, and right justified

Displays user's language instead of DB names

CLUES

1.     Clues are required for each field lable and each button that performs any sort of action. Flyover ToolTip help for each field and button.


2.     Flyover ToolTip for buttons should briefly describe what the button does without just being a repeat of the button label.


3.     Flyover ToolTips for fields should be a brief description of the purpose of the field, without simply restating the name of the field.


4.     Use extended tooltips (information icon with hover/click popup) for areas where users may have more difficulty using or understanding.
