CHECKBOXES & RADIO BUTTONS

INTRODUCTION

Input fields are used to collect user input. They can have multiple states: active, hover, disabled, which can have different background, text, and border colors. Use consistent spacing between rows of fields, and between the field and label text. Mark required fields with a red asterisk to the right of the field label.


Use Radio Buttons when a user has to choose only one of several values


Radio buttons should be lined up either in a single row or in a column


Labels for Radio buttons should be displayed to the right of the button.

CHECK BOXES

Use Check Boxes when a user can select one or more of several values


Check Boxes should be lined up either in a single row or in a column


Labels for Check Boxes should be displayed to the right of the check box.
