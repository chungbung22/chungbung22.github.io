LABELS AND FIELDS

LABELS

• Top-align labels with multi-line fields


• Right justify labels for fields read as columns


• Label all panels, tables, pages, columns, rows, input boxes.


• Display user’s language for labels rather than use database names


• Add object type to tab labels

FIELDS

Use consistent padding between controls with a minimum of 8px padding.


Use consistent spacing between rows of fields


Use a consistent style for fields (same corners for example)


Use consistent spacing between the field and the label text.


Mark required fields with an asterisk to the right of the field label.Use in-line helper text for short information or example values


Fields that are open for edit have a clear background.


Fields that are not open for edit have a light gray background.


On data-entry fields that are driven by a value list, the value will auto fill as the user types in a value.
