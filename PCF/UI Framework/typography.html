TYPOGRAPHY

3.25 INTRODUCTION

In an effort to modernize the look and feel of the BWS-DME UI, all (non-Unicode) UI elements will use a font from the Calibri family.



3.25.1 TEXT COLORS

The text color should be used to indicate the state of an object. The following states are used:


1.     Normal (black, #000000) - for normal text fields and labels


2.     Error (red, #FF0000) - for error message text


3.     Disabled (gray, #CCCCCC) - for disabled fields

3.25.2 SIZING

Type sizes should be used judiciously, not only to ensure consistency, but also to avoid clutter. These sizes were developed for optimal reading comfort and design content.


Heading One

Font-family : 'Calibri'

Font-size : 20px

Font-weight : bold


Used for Data Marking or title of popup such as BWS Colors


Heading Two

Font-family : 'Calibri'

Font-size : 15px

Font-weight : normal


Used for title of folders on homepage


Heading Three

Font-family : 'Calibri'

Font-size : 12px

Font-weight : bold


Used for title of form such as WBA Summary


Body Text

Font-family : 'Calibri'
