TITLE LINE & TEARAWAY TABS

3.5.1 TITLE LINE

The Title Line for the System includes the following information by default for the environment/model that has been logged into.

System Acro (BWS-DME)


Region Type shows only in Development and Test Environments.


Model number (747)


Sensitivity marking appropriate to the region/model See 02_11_02_Standard_Sensitivity_Marking



Standard buttons/icons for minizize, full-screen and exit

3.5.1 TEARAWAY TABS

The following features are standard for all Tear Away Tabs:


1.     Title Line on Tear-Away Tabs includes title of the form being torn-away:


        a.     Tear-Away tab Title replaces System Title and copies all other information from Title Line however a more specific sensitivity marking may be applied

                based on the data displayed in the form. See SENSITIVITY MARKINGS


        Example of tab EQ Drawing M11121 torn-away


2.     The form is sizable with bars both vertical and horzontial appearing once screen size has diminished beyond the display of the data to allow scrolling.


3.     Tear-away tabs are dockable back to the original session from which they were torn-away.


        a.     To re-dock the tab, position tab within the DME session and a shadow button appears that is used to attach tab back to the session.

        b.     Tabs may be docked into other un-docked tabs using the directional shadow button to determine how the window is split (horizontal vs. vertical).
