ICONS

3.13.2 INTRODUCTION

There are two types of toolbar icons, common and command. Common icons occupy the first row of the toolbar in the upper right hand corner. These are common functions to the tool that are not related to the page in focus. Command icons occupy the second row in the bottom left hand corner. These are functions related to the page in focus.


Icons should have a clear background and black text when active. When disabled, icons should still have a clear background, but the text and icons should be grayed out.
