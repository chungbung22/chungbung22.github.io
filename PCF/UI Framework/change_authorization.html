CHANGE AUTHORIZATION

3.22 INTRODUCTION

Change Authorization Numbers are used to authorize changes that occur to wire bundles during the workflow. This can be due to customer request when they wish to change the airplane configuration, or at the request of the engineering team for miscellaneous reasons. These are managed by the BWS-CM tool, hence the name change management, but can be viewed and set within the BWS-DME tool.


 Within a recognized drawing pattern, there will be a section to set the change authority. After selecting the set button, a form will popup allowing the user to search for the desired change authorization.

After selecting the desired change authorization number, the popup will close and the form will be populated with the selected change authority. It will be broken into four separate fields: 1) Change Number, 2) Part, 3) Supplement, and 4) GC.
