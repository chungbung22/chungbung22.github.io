Welcome to the BWS-DME UI FRAMEWORK

INTRODUCTION

The NWS Wiring user interface is a set of desktop ‘thick-client’ applications built using Windows Presentation Foundation (WPF)


The UI design consists of a drawing-tree and tab-based design that is based on existing UI frameworks for a number of Microsoft data-intensive editing products, including: Visual Studio 2010, SQL Server Management Studio, and Expression Blend 3.0.

 

The UI framework serves as a source of truth for all UI/pattern standards used in DME, and provides a set of standards/guidelines to follow.
