DEVELOPER-LEVEL SPECIFICATIONS

The NWS Wiring tool UI is built using Windows Presentation Foundation, a UI toolset for Windows .Net applications.


The UI toolset is very flexible and customizable.


All datasheets use the WFP Grid control, or the Infragistics grid control


The following section specifies developer-level specifications that are intended to inform the developer of the fundamental modules found in the DME-UI project.

In addition to providing a basic understanding on what the modules contain, this section features UML sequence diagrams that describe program flow for a certain set of scenarios.


In the future, IT needs to document more classes using UML class diagrams and capture additional run-time scenarios using UML sequence diagrams. Extensive documentation was not able to be performed as of 1/14/16 due to significant time constraints.
